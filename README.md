# Travel App Project

## Overview
Web application that displays weather and image of the location using information pulled from external APIs: Geonames, Weatherbit and Pixabay.

## Instructions
1. Run npm install
2. Run npm run build-dev | npm run build-prod
   Dev runs at port 8080, whereas prod runs at port 8000
3. Run npm run start
4. Create .env at the root directory and add the corresponding API keys.
    GEONAMES_USERID=******
    WEATHERBIT_API_KEY=******
    PIXABAY_API_KEY=******
5. Run npm test for unit testing
